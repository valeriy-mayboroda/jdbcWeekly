    Таблица user состоит из:
-id - Номер пользователя в БД (первичный ключ)
-firstname - Имя пользователя
-lastname - Фамилия пользователя
-age - Возвраст пользователя
-mail - Электронная почта пользователя
    Таблица meeting состоит из:
-id - Номер дела, встречи в БД (первичный ключ)
-name - Краткое название дела, встречи
-date - Дата дела, встречи
-time - Время дела, встречи
-details - Краткое описание дела, встречи
-userId - Номер пользователя для составленного дела, встречи (вторичный ключ)
    В main :
-1 - Вывод списка всех пользователей
-2 - Вывод списка дел, встреч всех пользователей
-3 - Вывод списка дел, встреч пользователя по id пользователя
-4 - Добавить нового пользователя
-5 - Добавить новое дело, встречу
-6 - Удалить дело, встречу по id дела, встречи
-7 - Удалить пользователя по id пользователя с удалением всех его дел, встреч
