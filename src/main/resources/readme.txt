    Таблица meeting состоит из:
-id - Номер события-встречи в БД
-user - Имя пользователя события-встречи
-name - Название события-встречи
-date - Дата события-встречи
-time - Время события-встречи
-details - Краткое пояснение события-встречи
    В main :
-1 - Вывод таблицы meeting в консоль
-2 - Добавление события-встречи в таблицу meeting с полями, вводимыми с консоли
-3 - Удаление события-встречи по вводимому с консоли id
